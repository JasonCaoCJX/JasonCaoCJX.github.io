## 字符串取值
python的字符串有2种取值顺序:

从左到右索引默认0开始的，最大范围是字符串长度少1

从右到左索引默认-1开始的，最大范围是字符串开头

可以使用 [头下标:尾下标] 来截取相应的字符串,包含头下标的字符，但不包含尾下标的字符

```
>>> s = 'python'
>>> s[1:5]
'ytho'
```

## 字符串连接
加号（+）是字符串连接运算符，星号（*）是重复操作

```
a = 'Hello'
b = 'World'
print(a + b)
print(a * 2)
```

结果为
```
HelloWorld
HelloHello
```

# 9. 异常处理
try-except构成的异常处理方法

掌握并熟练编写带有异常处理能力的程序，具备解决程序基本异常问题的能力

最基本的try-except-else结构
```
try:
    必须运行的代码
except:
    发生异常时执行的代码
else:
    没有异常时执行的代码
```

实例：
```
try:
    a = int(input("请输入一个整数："))
except:
    print("输入错误")
else:
    print(a)
```
如果输入数字1，输入正确，会执行第6行代码，得到的结果是：
```
1
```
如果字符串abc，输入错误，会执行第4行代码，得到的结果是：
```
输入错误
```

import turtle as t    # 导入turtle包
t.down()              # 开始制图（默认初始方向为正右）

	t.forward(100)    # 向前100像素
	t.right(72)       # 向右旋转72度

t.done()              # 完成制图
